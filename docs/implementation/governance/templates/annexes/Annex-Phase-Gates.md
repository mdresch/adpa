# Annex D – Phase‑Gate Checklists

> Evidence packs must be complete and green to proceed; store each gate decision in /onboarding/snapshots/. 

## Gate 0 – Charter & Funding
- Business case approved, sponsor named, high-level scope & timeline, benefits hypothesis

## Gate 1 – Requirements & Design
- Baseline requirements, signed design, integration contracts, security & privacy patterns

## Gate 2 – Build Complete
- Unit/Integration test coverage, code review reports, data model freeze

## Gate 3 – Test Complete
- UAT sign-off, performance test pass, defect leakage within threshold

## Gate 4 – Go/No‑Go
- Cutover plan, rollback plan, support model, SteerCo approval

## Gate 5 – Hypercare Exit
- Adoption metrics on track, incident levels normalized, benefits tracking in flight
