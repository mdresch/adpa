# 🛠️ ADPA Feature Documentation

This directory contains detailed documentation for each feature of the ADPA platform.

## 🗂️ Feature Categories

- **[Document Processing](document-processing/)**: Core document ingestion, OCR, and extraction.
- **[AI Engine](ai-engine/)**: LLM orchestration, prompt management, and observability (Langfuse).
- **[Workflow](workflow/)**: Human-in-the-loop, task automation, and audit logging.
- **[Analytics](analytics/)**: Dashboards, reporting, and trend analysis.
- **[Integrations](integrations/)**: Connectors for eSignature, Digital Twins, and external systems.
- **[Compliance](compliance/)**: Data governance, GDPR/CCPA, and security best practices.
- **[Legacy 06 Features](legacy-06-features/README.md)**: Migrated documentation from the previous `docs/06-features` structure.

---

### 📝 How to Use This Guide

1.  **For developers**: Start with `AI Engine` and `Document Processing`.
2.  **For product owners**: Start with `Feature List` (`../feature-list.md`) and `Integrations`.
3.  **For compliance officers**: Review `Compliance` and `Workflow/Audit.md`.

> 📌 *Tip: All documentation is built using Markdown and is GitHub-flavored.*
