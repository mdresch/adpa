# 🧩 Parsing (OCR & Layout Analysis)

Convert raw documents into structured data.

## Key Technologies

- **OCR Engine**: Tesseract or commercial equivalent (e.g., Azure Form Recognizer).
- **Layout Analysis**: LayoutLM or Donut for table and section detection.
- **Text Normalization**: Handle font sizes, bold, and alignment to infer hierarchy.

## ⚙️ Output

- **Raw Text**: Clean, searchable text.
- **Structured Layout**: Blocks of text with metadata (e.g., `heading`, `paragraph`, `table`).
- **Annotations**: Highlighted form fields and signatures (for review).
