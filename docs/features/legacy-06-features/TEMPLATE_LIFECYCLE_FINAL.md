## 📋 **Compliance Checklist (PMBOK 7)**

When your template reaches Compliance stage, reviewer will check:

**Document Structure**:
- [ ] Project title clearly stated
- [ ] Project manager & sponsor identified
- [ ] Objectives listed and specific
- [ ] Scope includes in/out boundaries
- [ ] Key deliverables defined
- [ ] Stakeholder register present
- [ ] High-level schedule included
- [ ] Budget summary provided
- [ ] Risk assessment included
- [ ] Success criteria defined
- [ ] Assumptions documented
- [ ] Constraints identified
- [ ] Approval section present

**PMBOK 7 Alignment**:
- [ ] Terminology follows PMBOK 7
- [ ] Process groups addressed
- [ ] Knowledge areas covered
- [ ] Tailoring explained (if applicable)

**Quality**:
- [ ] Professional tone
- [ ] Clear and concise
- [ ] Actionable content
- [ ] Consistent formatting (Markdown)

**Score**: Aggregate → Compliance Score (0-100%)

**Minimum**: 80% to promote to Validated
