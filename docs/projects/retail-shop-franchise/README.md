## Program Pillars

- **Microsoft Experience Centers** — Experiential retail, hands-on discovery, workshops.
- **Microsoft Innovation Hub** — Demos, co-creation, Azure AI/IoT/Digital Twins.
- **Customer Engagement Programs** — Workshops, training, adoption, ongoing engagement.

## Digital Twin

The layout and sensor design support a **Digital Twin** of the shop (Azure IoT / Azure Digital Twins). Asset types include structural, device, sensor/IoT, experience, environmental, and customer interaction nodes. See [01-project-ideation §5](./01-project-ideation.md#5-digital-twin-architecture) and [03-shop-layout §5](./03-shop-layout-customer-flow-sensors.md#5-digital-twin-alignment).
