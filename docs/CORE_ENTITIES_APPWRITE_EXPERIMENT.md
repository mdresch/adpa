# Core Entities for Appwrite Experiment

## Core Entity Types (Tier 1)
Based on the ExtractionRegistry analysis, these are the essential entities:

### Project Fundamentals
- **stakeholders** - People and organizations involved in the project
- **requirements** - Project requirements and specifications
- **risks** - Project risks and issues
- **milestones** - Key project milestones
- **deliverables** - Project deliverables and outputs

### Project Execution
- **activities** - Project activities and tasks
- **scope_items** - Scope items and work breakdown
- **success_criteria** - Project success criteria
- **constraints** - Project constraints and limitations

### Resource Management
- **resources** - Project resources (human, equipment, etc.)
- **technologies** - Technologies used in the project

### Quality & Standards
- **quality_standards** - Quality standards and compliance
- **best_practices** - Best practices and guidelines
