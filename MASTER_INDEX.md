## 🎯 **Usage Cheat Sheet**

### **Agent 3 Commands:**
```bash
# View quality dashboard
http://localhost:3000/admin/quality-trends

# Run migrations
cd server && npm run migrate:agent3

# Test optimization
npm run test:template-optimization

# Check migration status
npm run migrate:status
```

### **Automation Commands:**
```bash
# Setup (one-time)
cd scripts/issue-automation && npm install

# Create briefing
npm run create

# Validate
npm run validate ../../YOUR_BRIEFING.md

# Preview
npm run preview ../../YOUR_BRIEFING.md
```
